# Branding — Claiire-App

## Nom

**Claiire** — lié au site claiire.com, hommage à Claire. Le double "i" est distinctif et mémorable. Se prononce comme le prénom, mais s'écrit différemment pour marquer l'identité.

---

## Taglines

- **Principal** : "Ton allié au quotidien"
- **Warrior** : "Combats tes démons, gagne ta vie"
- **Store description** : "Un compagnon IA 100% privé pour reprendre le contrôle"

---

## Valeurs de marque

| Valeur | Signification |
|---|---|
| **Discrétion** | L'app ne trahit jamais — pas de notifications gênantes, pas de données envoyées, icône neutre |
| **Force intérieure** | L'utilisateur a déjà tout en lui, Claiire l'aide à le révéler |
| **Progression** | Chaque pas compte, même les petits — le système récompense la constance |
| **Non-jugement** | Zéro culpabilisation, même en cas de rechute — "un nouveau combat commence" |

## Identité visuelle

### Mode sombre par défaut

Le mode sombre est le mode principal. Il est :
- **Apaisant** : réduit la stimulation visuelle
- **Discret** : moins visible si quelqu'un regarde par-dessus l'épaule
- **Économe** : meilleure autonomie sur les écrans OLED

### Palette de couleurs

#### Couleurs de base
- **Background principal** : Noir profond (#0A0A0F) à gris très sombre (#12121A)
- **Surfaces** : Gris sombre (#1A1A2E) pour les cartes et conteneurs
- **Texte principal** : Blanc cassé (#E8E8F0)
- **Texte secondaire** : Gris moyen (#8888A0)

#### Couleurs d'accent (gamification)
- **XP / Or** : Doré (#FFD700) — récompenses, progression, étoiles
- **Victoire / Succès** : Vert émeraude (#00D68F) — streaks, accomplissements, validation
- **Combat / Énergie** : Rouge-orange (#FF6B35) — mode Warrior, batailles, urgence
- **Premium** : Violet profond (#7C3AED) — fonctionnalités premium, contenu exclusif

#### Couleurs des compagnons
- **Lumo** : Bleu glacier (#4FC3F7) — calme, stratégie, clarté
- **Papillon** : Jaune vif (#FFE066) — énergie, joie, hype
- **Étoile** : Violet doux (#B39DDB) — douceur, apaisement, réconfort
- **Sage** : Vert sauge (#81C784) — analyse, sagesse, équilibre (Phase 2)
- **Aurore** : Orange chaud (#FFB74D) — motivation, chaleur, élan (Phase 2)

### Dual Mode UI

#### Mode Warrior
- Background : Noir profond avec des accents rouges/orange
- Ambiance : Intense, déterminée, combative
- Animations : Plus dynamiques, particules, effets de combat
- Icônes : Épée, bouclier, flamme

#### Mode Zen
- Background : Bleu nuit profond avec des accents bleus/verts
- Ambiance : Calme, introspective, apaisante
- Animations : Lentes, fluides, respirantes
- Icônes : Feuille, eau, lune

## Les 5 compagnons

### MVP (Phase 1)

#### Lumo — Le Stratège
- **Personnalité** : Calme, posé, réfléchi. Parle avec précision.
- **Rôle** : Aide à planifier, analyser les situations, préparer les combats
- **Palette** : Bleus froids, argenté
- **Expression Rive** : Yeux pensifs, mouvements mesurés
- **Phrase type** : "Analysons la situation ensemble."

#### Papillon — Le Hype Man
- **Personnalité** : Énergique, enthousiaste, explosif. Parle avec des exclamations.
- **Rôle** : Célèbre les victoires, booste le moral, motive dans les moments durs
- **Palette** : Jaunes vifs, dorés
- **Expression Rive** : Yeux brillants, mouvements rapides et joyeux
- **Phrase type** : "Tu gères, champion !"

#### Étoile — Le Guérisseur
- **Personnalité** : Doux, apaisant, empathique. Parle doucement.
- **Rôle** : Réconforte, écoute, apaise dans les moments difficiles
- **Palette** : Violets doux, lavande
- **Expression Rive** : Yeux bienveillants, mouvements lents et fluides
- **Phrase type** : "Je suis là, prends ton temps."

### Phase 2

#### Sage — L'Analyste
- **Personnalité** : Analytique, curieux, méthodique
- **Rôle** : Données, tendances, insights sur la progression
- **Palette** : Verts sauge, tons naturels

#### Aurore — Le Motivateur
- **Personnalité** : Chaleureux, déterminé, inspirant
- **Rôle** : Fixation d'objectifs, défis ambitieux, dépassement de soi
- **Palette** : Oranges chauds, tons soleil levant

## Typographie

- **System fonts** : Pas de polices custom pour maximiser les performances mobile
  - iOS : SF Pro (San Francisco)
  - Android : Roboto
- **Hiérarchie** :
  - Titres : Bold, taille large
  - Corps : Regular, taille confortable (16px minimum)
  - Accents : Semi-bold pour les données importantes (XP, streaks)

---

## Iconographie

- **Bibliothèque** : Lucide Icons — cohérente, lisible, open source
- **Style** : Outline (trait fin), cohérent avec le mode sombre
- **Icônes spécifiques Claiire** :
  - Épée (missions Warrior)
  - Bouclier (protection, streak)
  - Flamme (streak actif)
  - Étoile (XP gagné)
  - Cœur (mode Zen, bien-être)

## Présence Store

### Icône de l'app
- Discrète et neutre — ne doit pas trahir le contenu si quelqu'un voit le téléphone
- Éviter tout symbole trop explicite (pas de cerveau, pas de chaîne brisée)
- Suggestion : le "C" de Claiire stylisé, ou un des compagnons simplifié

### Screenshots Store
1. Compagnon IA en conversation (mettre en avant la relation)
2. Mode Warrior — tableau de bord combat
3. Mode Zen — journal et habitudes
4. Système de progression (XP, niveaux, streaks)
5. Message "100% privé — IA on-device"

### Description courte (30 caractères)
"Ton allié bien-être gamifié"

### Description longue
Mettre en avant dans cet ordre :
1. Confidentialité (100% on-device)
2. Les compagnons IA personnalisés
3. La gamification (combat system)
4. Les deux modes (Warrior / Zen)
5. Gratuit pour démarrer
