[(157) How I built a simple RPG game to track my life and skills - YouTube](https://www.youtube.com/watch?v=GMn9sxCWN0M)

- 🎮 Création d'une application RPG: L'auteur a développé une application RPG pour suivre la progression et acquérir des compétences en utilisant des points d'expérience et de niveau.

- 🧠 Inspiration des jeux: L'application intègre des éléments de jeux populaires comme Pokémon, Dragon Ball Z et Final Fantasy pour stimuler l'apprentissage et le retour d'expérience des utilisateurs.

- 📱 Conception UI/UX: La conception initiale a été développée sur un iPad et transférée sur Figma, en optimisant l'interface pour une utilisation mobile.

- ☁️ Intégration avec Astra DB: L'application utilise Astra DB, un service de base de données sur Apache Cassandra, pour gérer les données de l'utilisateur, sans besoin de configuration complexe.

- 💻 Développement React: L'application fonctionne sur Create React App avec un design mobile-first, utilisant Tailwind CSS pour une expérience utilisateur fluide.

- 📡 Fonctionnalités sans serveur: Utilisation de Netlify pour gérer le backend de manière serverless, en facilitant l'accès aux API endpoints et à la base de données.

- 🛠️ Gestion des compétences en temps réel: Ajouter, modifier et supprimer des compétences, mises à jour en temps réel dans la base de données et sur l'écran de l'utilisateur.

- 📲 Accessibilité et partage: L'application est accessible via mobile et peut être partagée sur GitHub pour permettre aux utilisateurs de la tester et contribuer aux améliorations.
